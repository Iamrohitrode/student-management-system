# 🎓 LearnX - Student Management System

A sleek and modern student dashboard designed to manage academic life efficiently — from attendance tracking to assignments and notes — all in one place.

---

## 🌟 Features

- 🔐 Student Registration & Login
- 📊 Dashboard with real-time insights
- 📅 Timetable Management
- ✅ Task & Assignment Tracker
- 📝 Notes / Study Vault
- 📈 Attendance Monitoring
- 👤 Profile Management with image upload

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript
- LocalStorage (for data persistence)
